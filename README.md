# Discord GIF Reaction Bot

Бот для Discord, написанный на Python с использованием библиотеки `discord.py`. Автоматически реагирует на определённые гифки (по URL или провайдеру Tenor) и отправляет случайные картинки из локальной папки `images`. Также предоставляет команду `!image` для ручной отправки случайной картинки.

## 🚀 Функции

- **Автоматическая реакция на гифки**: Если в сообщении содержится целевой URL гифки (настраивается через переменную окружения `TARGET_GIF_URL`) или embed от провайдера Tenor, бот отправляет случайную картинку из папки `images`.
- **Ручная отправка картинок**: Команда `!image` отправляет случайную картинку из локальной папки.
- **Логирование embed'ов**: В консоли выводится детальная информация о каждом embed'е в сообщениях (для отладки и настройки).
- **Поддержка локальных изображений**: Все картинки хранятся локально в папке `images` (форматы: .jpg, .png, .gif и др.).

## 📦 Установка и настройка

### 1. Клонирование репозитория
```bash
git clone https://github.com/CEPIbN/Discord-bot.git
```
```bash
cd Discord-bot
```

### 2. Установка зависимостей
Убедитесь, что у вас установлен Python 3.13 или выше. Затем установите зависимости:
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Создайте файл `.env` в корне проекта и добавьте следующие переменные:
```env
BOT_TOKEN=your_discord_bot_token_here
TARGET_GIF_URL=https://example.com/target.gif  # опционально, URL гифки-триггера
```
- **BOT_TOKEN**: Токен вашего Discord-бота. Получить можно на [Discord Developer Portal](https://discord.com/developers/applications).
- **TARGET_GIF_URL**: (Опционально) URL гифки, на которую бот будет реагировать. Если не задан, бот будет реагировать только на embed'ы от Tenor и гифки с ID `25572384`.

### 4. Подготовка папки с картинками
Создайте папку `images` в корне проекта и поместите туда любые изображения (поддерживаемые Discord форматы: JPG, PNG, GIF и т.д.):
```bash
mkdir images
```
Бот автоматически загрузит все файлы из этой папки при запуске.

## 🎮 Использование

### Запуск бота
```bash
python bot.py
```
При успешном запуске в консоли появится сообщение:
```
Бот <имя_бота> успешно запущен!
Загружено <N> локальных картинок.
```

### Команды
- `!image` – отправляет случайную картинку из папки `images` в текущий канал.

### Автоматические реакции
1. **Реакция на текст**: Если сообщение содержит URL, указанный в `TARGET_GIF_URL`, бот отправит случайную картинку.
2. **Реакция на embed'ы**: Бот анализирует все embed'ы в сообщении. Если обнаружен embed с URL, содержащим `25572384`, или провайдером `Tenor`, бот также отправит случайную картинку.

### Отладка
В консоли выводятся подробные данные о каждом embed'е (тип, URL, заголовок, описание и т.д.). Это помогает настроить триггеры под нужные гифки.

## ⚙️ Конфигурация

Настройки бота задаются в коде `bot.py`:

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `TARGET_GIF_URL` | URL гифки-триггера (из переменной окружения) | `None` |
| `IMAGES_FOLDER` | Папка с локальными картинками | `"images"` |
| `BOT_TOKEN` | Токен бота (из переменной окружения) | – |
| `COMMAND_PREFIX` | Префикс команд | `!` |

Вы можете изменить эти параметры, отредактировав `bot.py`.

## 🔧 Возможные проблемы и решения

- **Бот не запускается**: Убедитесь, что токен бота указан верно и у бота есть необходимые права (например, `Send Messages`, `Read Message Content`).
- **Картинки не отправляются**: Проверьте, что папка `images` существует и содержит файлы. Также убедитесь, что бот имеет право отправлять файлы в канал.
- **Бот не реагирует на гифки**: Проверьте, что переменная `TARGET_GIF_URL` задана корректно, либо что embed содержит нужный URL или провайдера. Используйте вывод отладки в консоли для анализа embed'ов.
